About the Project

StegoSecure helps hide sensitive information inside images—quietly, cleanly, and professionally—using LSB steganography with optional encryption.

What is Steganography

It’s the art of hiding data inside another file so the existence of the message is hard to detect.

Why Security is Important

Steganography reduces attention. With encryption, it also protects content even if extracted.

Our Project Objective

Offer a polished, examiner-ready UI for encoding/decoding multiple file types inside PNG images.
